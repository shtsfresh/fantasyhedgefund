# Fantasy Hedge Fund

#### Setup
1. Install nix: ```https://nixos.org/download.html```
2. Install docker desktop: ```https://www.docker.com/products/docker-desktop/```
3. Update development environment:
```sh
./bin/fantasy.sh 
```

#### Build
```sh
./bin/fantasy.sh build
```

#### Start
```sh
./bin/fantasy.sh start
```
